# chatbot
<p> This is a simple chatbot that explains what is credit score and gives advice on how to save.</p>
<h3>Running the application</h3>
<p>To run the application one needs to have installed python on his/her computer</p>
<p>Clone/download this repository</p>
<p>Run pip install -r requirements.txt on cmd to intall all needed modules for the program to run</p>
<p>Run the program on the cmd or on any python editor available to you</p>
<h3>Commands you can try</h3>
<p>What is credit score</p>
<p>What is credit score determined by</p>
<p>Tips on how to save</p>
<p>What is the largest factor that determines my credit score<p>
<p>Ways to improve my credit score</p>
<p>Who is a responsible borrower</p>
<p>And any other questions you may have about credit score and saving</p>
<p>Enjoy!</p>
